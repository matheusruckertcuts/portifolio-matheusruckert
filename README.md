# Portfólio — Matheus Rückert

Site pessoal de portfólio, editor de vídeos curtos (Reels, TikTok, Shorts).

## Como publicar no GitHub Pages (grátis, ~3 minutos)

1. Crie uma conta em [github.com](https://github.com) (se ainda não tiver).
2. Clique em **New repository** (botão verde "New").
3. Nome do repositório: `portfolio` (ou o nome que preferir).
4. Marque como **Public**. Não marque nenhuma outra opção. Clique em **Create repository**.
5. Na página do repositório recém-criado, clique em **uploading an existing file**.
6. Arraste os arquivos `index.html` e `.nojekyll` desta pasta para dentro da área de upload.
7. Clique em **Commit changes** (botão verde, embaixo).
8. Vá em **Settings** (menu do topo do repositório) → **Pages** (menu lateral esquerdo).
9. Em "Build and deployment" → **Source**, selecione **Deploy from a branch**.
10. Em **Branch**, selecione `main` e a pasta `/ (root)`. Clique em **Save**.
11. Aguarde 1-2 minutos. Atualize a página — vai aparecer um link no topo:
    `https://SEU-USUARIO.github.io/portfolio/`

Esse é o link do seu site, pronto para compartilhar com clientes, colocar na bio do Instagram, TikTok, WhatsApp Business, etc.

## Atualizando o site depois

Sempre que quiser trocar algo (novo case, novo número, novo contato):
1. Vá até o repositório no GitHub.
2. Clique no arquivo `index.html`.
3. Clique no ícone de lápis (Edit) ou apague e suba a versão nova pelo mesmo processo de upload.
4. Commit changes — o site atualiza sozinho em 1-2 minutos.

## Arquivos

- `index.html` — o site completo (HTML, CSS e JS em um único arquivo, com todas as imagens e vídeos já embutidos).
- `.nojekyll` — arquivo técnico que evita processamento desnecessário pelo GitHub Pages (não precisa mexer nele).
